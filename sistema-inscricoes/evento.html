Uma LP pronta para eventos paralelos

Esta pagina busca o evento pelo identificador informado na URL. Isso permite ter mais de uma landing page ativa ao mesmo tempo, cada uma apontando para o seu proprio cadastro dentro do sistema.

01
Evento vinculado automaticamente

O formulario grava a inscricao diretamente em inscricoes_eventos usando o id do evento carregado.

02
Ficha alinhada ao fluxo do retiro

A pagina coleta nome, telefone, email, rede e se a pessoa ja foi para o Retiro.

03
Visual inspirado na arte enviada

Mantivemos a direcao preto, branco e laranja com uma composicao mais forte para pagina de conversao.
